About this page

This page is a small program allowing you to define an HTML document and a JavaScript file, then invoke the script within the context of the HTML document.

Documents created can be saved to browser local storage. The "Run Link" and "Load Link" can be used to create bookmarks that run or just load your code. The "Run Link" embeds an access token so that only those knowing the link can cause your browser to run the code.
